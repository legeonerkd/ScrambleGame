# ScrabbleGame (Эрудит)

Настольная игра "Эрудит" (Scrabble) на русском языке с графическим интерфейсом на Python/Tkinter.

## Описание

ScrabbleGame - это реализация классической игры "Эрудит", где игроки составляют слова из букв на игровом поле размером 15x15 клеток. Игра поддерживает режимы PvP (игрок против игрока) и PvE (игрок против AI).

## Возможности

- 🎮 Два режима игры: PvP и против AI
- 📚 Словарь русских слов (~33,800 слов, минимум 3 буквы)
- ➕ **Добавление пользовательских слов в словарь**
- 🎯 Система бонусов (множители букв и слов)
- 🔄 Замена букв (с кулдауном)
- 🏆 Подсчет очков
- 🤖 AI противник "Степан"
- ✅ Модульное тестирование (66 тестов)

## Требования

- Python 3.6+
- tkinter (обычно входит в стандартную установку Python)

## Правила игры

1. **Первый ход**: Первое слово должно проходить через центральную клетку (7,7)
2. **Минимальная длина слова**: Все слова должны содержать минимум 3 буквы
3. **Размещение букв**: 
   - Слова должны быть горизонтальными или вертикальными
   - ⚠️ **НЕЛЬЗЯ** ставить буквы на уже занятые клетки
   - Буквы в составленных словах не могут быть заменены
4. **🎯 Классическое правило Scrabble**: Каждое новое слово должно использовать хотя бы одну букву из уже размещенных на доске (примыкать к существующим словам)
5. **Валидация**: Все слова (основные и побочные) проверяются по словарю
5. **Очки**: Каждая буква имеет свою стоимость (А=1, Ф=10, и т.д.)
6. **Бонусы**: 
   - Центральная клетка: x2 к слову
   - 10 случайных бонусных клеток: x2 или x3 к букве/слову
   - Бонусы применяются только один раз
7. **Замена букв**: Можно заменить до 3 букв раз в 3 хода
8. **Бонус за 7 букв**: +50 очков за использование всех букв за один ход

## Управление

- **Выбор буквы**: Клик по букве на стойке
- **Размещение**: Клик по клетке на поле
- **Подтверждение хода**: Кнопка "✓ Ход"
- **Отмена**: Кнопка "↩ Отмена"
- **Замена букв**: Кнопка "🔁 Замена" (выберите 3 буквы)
- **Добавление слова**: Кнопка "📖 Добавить слово" (расширение словаря)

## Известные ограничения

- AI размещает слова только в центре поля
- Нет сохранения/загрузки игры
- Нет истории ходов
- Нет drag & drop для букв
- Бонусные клетки генерируются случайно (не как в оригинальном Scrabble)

## Пользовательские слова

Игра поддерживает добавление собственных слов в словарь:

1. Нажмите кнопку **"📖 Добавить слово"** во время игры
2. Введите новое слово (минимум 3 буквы, только русские буквы)
3. Слово автоматически сохранится в файл `custom_words.txt`
4. Все добавленные слова будут доступны в следующих играх

### Требования к словам:
- ✅ Минимум 3 буквы
- ✅ Только русские буквы (А-Я, Ё)
- ✅ Слово не должно быть в основном словаре
- ✅ Нет дубликатов

```bash
# Запустить тесты словаря
pytest tests/test_dictionary.py -v

# Запустить с покрытием
pytest tests/ --cov=models --cov-report=html
```
