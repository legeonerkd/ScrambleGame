# Пользовательские слова в ScrabbleGame

## Обзор

ScrabbleGame версии 2.1+ поддерживает добавление пользовательских слов в словарь. Это позволяет расширять игровой словарь, добавляя редкие, специфические или жаргонные слова.

## Как добавить слово

### Через интерфейс игры

1. Запустите игру: `python main.py`
2. Нажмите кнопку **"📖 Добавить слово"**
3. В открывшемся окне введите слово
4. Нажмите **"Добавить"**

![Добавление слова](docs/add_word_screenshot.png)

### Требования к словам

Слово должно соответствовать следующим требованиям:

| Правило | Описание | Пример |
|---------|----------|--------|
| ✅ Минимум 3 буквы | Слово должно содержать не менее 3 символов | ✅ КОТ, ❌ АХ |
| ✅ Только кириллица | Только русские буквы А-Я, Ё | ✅ ЁЖИК, ❌ TEST |
| ✅ Нет дубликатов | Слово не должно быть в словаре | ✅ НОВОЕСЛОВО, ❌ КОТ |
| ✅ Без пробелов | Слово пишется слитно | ✅ САМОЛЁТ, ❌ САМО ЛЁТ |
| ✅ Без цифр | Только буквы | ✅ РОБОТ, ❌ РОБОТ2 |

### Примеры

#### ✅ Валидные слова
```
ГАДЖЕТ
ИНТЕРНЕТ
БЛОГЕР
СЕЛФИ
ХАЙП
```

#### ❌ Невалидные слова
```
АЙ         # Слишком короткое (минимум 3 буквы)
ROBOT      # Латиница вместо кириллицы
ХАЙП2      # Содержит цифру
КОТ        # Уже есть в основном словаре
```

## Хранение слов

### Файл custom_words.txt

Пользовательские слова хранятся в файле `custom_words.txt` в директории игры:

```
ScrabbleGame/
├── words.txt           # Основной словарь (33,772 слова)
└── custom_words.txt    # Пользовательские слова
```

Формат файла - по одному слову на строке в верхнем регистре:

```
ГАДЖЕТ
ИНТЕРНЕТ
БЛОГЕР
СЕЛФИ
```

### Ручное редактирование

Вы можете редактировать `custom_words.txt` вручную:

1. Откройте файл в текстовом редакторе
2. Добавьте слова (по одному на строку, в верхнем регистре)
3. Сохраните файл в кодировке UTF-8
4. Перезапустите игру

**Важно**: Соблюдайте правила (минимум 3 буквы, только кириллица)!

## Программный интерфейс

### Добавление слова через API

```python
from models.dictionary import ScrabbleDictionary

# Создаем словарь
dictionary = ScrabbleDictionary()

# Добавляем слово
success, message = dictionary.add_word("НОВОЕСЛОВО")

if success:
    print(f"✅ {message}")
    # Сохраняем в файл
    dictionary.save_custom_words()
else:
    print(f"❌ {message}")
```

### Проверка слова

```python
# Проверяем наличие слова (основной + пользовательский словарь)
if dictionary.is_word("ГАДЖЕТ"):
    print("Слово есть в словаре")

# Оператор 'in'
if "ИНТЕРНЕТ" in dictionary:
    print("Слово есть в словаре")
```

### Статистика

```python
# Общее количество слов
total = len(dictionary)

# Только пользовательские слова
custom_count = dictionary.get_custom_words_count()

print(f"Основной словарь: {len(dictionary.words)} слов")
print(f"Пользовательский: {custom_count} слов")
print(f"Всего: {total} слов")
```

## Валидация

### Автоматическая проверка

При добавлении слова автоматически проверяется:

1. **Длина**: минимум 3 символа
2. **Алфавит**: только А-Я, Ё
3. **Уникальность**: нет в основном словаре
4. **Дубликаты**: не добавлено ранее

### Сообщения об ошибках

| Ошибка | Причина | Решение |
|--------|---------|---------|
| "Слово не может быть пустым" | Пустая строка | Введите слово |
| "Слово должно содержать минимум 3 буквы" | < 3 символов | Используйте слова от 3 букв |
| "Слово должно содержать только русские буквы" | Латиница/цифры | Только А-Я, Ё |
| "Слово 'X' уже есть в основном словаре" | Дубликат | Слово уже доступно |
| "Слово 'X' уже есть в пользовательском словаре" | Повторное добавление | Слово уже добавлено |

## Резервное копирование

### Экспорт пользовательских слов

```bash
# Создать резервную копию
cp custom_words.txt custom_words_backup.txt

# Или с датой
cp custom_words.txt custom_words_$(date +%Y%m%d).txt
```

### Импорт на другой компьютер

1. Скопируйте `custom_words.txt` в директорию игры
2. Убедитесь что кодировка UTF-8
3. Запустите игру - слова автоматически загрузятся

## Лучшие практики

### ✅ Рекомендуется

- Добавлять узкоспециализированные термины
- Современные слова (гаджет, селфи, блогер)
- Региональные слова
- Профессиональный жаргон
- Редкие слова, которых нет в словаре

### ❌ Не рекомендуется

- Жаргонизмы сомнительной валидности
- Сленг с латиницей
- Слова с ошибками
- Несуществующие слова
- Аббревиатуры из латиницы (IT, PC)

## Часто задаваемые вопросы

**Q: Можно ли добавлять слова с буквой Ё?**  
A: Да, буква Ё поддерживается.

**Q: Сколько слов можно добавить?**  
A: Неограниченно, но рекомендуется до 1000 для производительности.

**Q: Будут ли слова доступны после перезапуска?**  
A: Да, слова сохраняются в `custom_words.txt` и загружаются автоматически.

**Q: Можно ли удалить добавленное слово?**  
A: Да, отредактируйте `custom_words.txt` вручную и удалите строку со словом.

**Q: Слово не принимается в игре, хотя я его добавил**  
A: Перезапустите игру - слова загружаются при старте.

## Поддержка

При проблемах с добавлением слов:

1. Проверьте логи в консоли
2. Убедитесь что файл `custom_words.txt` в кодировке UTF-8
3. Проверьте что слово соответствует всем требованиям
4. Создайте issue на GitHub с описанием проблемы
