# Правила игры ScrabbleGame

## Основные правила

### 1. Цель игры
Набрать максимальное количество очков, составляя слова из букв на игровом поле 15x15 клеток.

### 2. Начало игры
- Каждый игрок получает 7 случайных букв
- В стойке не может быть более 2 одинаковых букв
- Первый ход делает Игрок 1

### 3. Правила составления слов

#### ⭐ Первый ход
- Первое слово ОБЯЗАТЕЛЬНО должно проходить через центральную клетку (7,7)
- Слово должно содержать минимум 3 буквы

#### 📏 Минимальная длина слова
- **Все слова должны содержать минимум 3 буквы**
- Это относится как к основному слову, так и к побочным словам
- Слова из 1-2 букв не принимаются

#### 🔒 Защита составленных слов
- **НЕЛЬЗЯ ставить новые буквы на уже занятые клетки**
- Буквы в составленных словах не могут быть заменены или перемещены
- Можно только добавлять буквы рядом с существующими словами

#### 🎯 Классическое правило Scrabble (соединение)
- **Каждое новое слово ОБЯЗАТЕЛЬНО должно использовать существующие буквы**
- Новое слово должно примыкать хотя бы к одной букве из уже размещенных
- Нельзя размещать изолированные слова отдельно от основного игрового поля
- Исключение: первый ход (доска пустая)

#### 📐 Ориентация слов
- Слова должны быть строго горизонтальными или вертикальными
- Диагональные слова запрещены

#### ✅ Валидация слов
- Все слова проверяются по словарю (~33,800 русских слов)
- Проверяются как основное слово, так и все побочные слова
- Если хотя бы одно слово невалидно - ход отклоняется

### 4. Подсчет очков

#### Стоимость букв
| Очки | Буквы |
|------|-------|
| 1 | А, В, Е, И, Н, О, Р, С, Т |
| 2 | Д, К, Л, М, П, У |
| 3 | Б, Г, Ё, Ь, Я |
| 4 | Й, Ы |
| 5 | Ж, З, Х, Ц, Ч |
| 8 | Ш, Э, Ю |
| 10 | Ф, Щ, Ъ |

#### Бонусные клетки
- **Центр (7,7)**: x2 к слову (используется только один раз)
- **10 случайных клеток**: x2 или x3 к букве или слову
- Бонусы применяются только при первом использовании клетки
- Если на клетке с бонусом уже стоит буква, бонус не действует

#### Бонус за 7 букв
- **+50 очков** за использование всех 7 букв за один ход

#### Расчет очков
1. Считается сумма очков всех размещенных букв
2. Применяются бонусы к буквам (x2 или x3 к букве)
3. Применяются бонусы к слову (x2 или x3 ко всей сумме)
4. Если использованы все 7 букв - добавляется +50 очков

Пример:
```
Слово "КОТ" на пустом поле без бонусов:
К(2) + О(1) + Т(1) = 4 очка

Слово "КОТ" с бонусом x2 к букве К:
К(2×2) + О(1) + Т(1) = 6 очков

Слово "КОТ" с бонусом x2 к слову:
(К(2) + О(1) + Т(1)) × 2 = 8 очков
```

### 5. Замена букв

- Можно заменить до **3 букв** за один ход
- Замена доступна **раз в 3 хода**
- При замене:
  1. Выбранные буквы возвращаются в мешок
  2. Игрок получает новые буквы из мешка
  3. Ход переходит к другому игроку

### 6. Пополнение стойки

- После каждого хода стойка автоматически пополняется до 7 букв
- Буквы берутся случайным образом из мешка
- Если в мешке недостаточно букв, игрок получает оставшиеся

### 7. Конец игры

Игра заканчивается когда:
- В мешке закончились буквы И
- У одного из игроков не осталось букв в стойке И
- Невозможно больше составлять слова

Побеждает игрок с наибольшим количеством очков.

## Примеры ходов

### ✅ Валидные ходы

#### Пример 1: Первый ход
```
Размещаем "СЛОВО" горизонтально через центр:
Позиции: (7,6), (7,7), (7,8), (7,9), (7,10)
Буквы: С, Л, О, В, О
✅ Проходит через центр (7,7)
✅ Минимум 3 буквы
✅ Слово есть в словаре
```

#### Пример 2: Второй ход
```
Есть слово "КОТ" на позициях (7,7), (7,8), (7,9)
Добавляем "ИК" вертикально:
Позиции: (6,9), (8,9)
Получается слово "ТИК" через Т из "КОТ"
✅ Не ставим на занятые клетки
✅ Создается слово из 3+ букв
✅ Все слова валидны
```

### ❌ Невалидные ходы

#### Пример 1: Слишком короткое слово
```
Размещаем "ОН":
❌ Ошибка: "Слово должно содержать минимум 3 буквы"
```

#### Пример 2: Замена букв в слове
```
Есть слово "КОТ" на позициях (7,7), (7,8), (7,9)
Пытаемся поставить Д на позицию (7,8):
❌ Ошибка: "Клетка (7,8) уже занята буквой 'О'"
```

#### Пример 3: Побочное слово из 2 букв
```
Создается побочное слово "АХ":
❌ Ошибка: "Побочное слово «АХ» слишком короткое (минимум 3 буквы)"
```

## Стратегия игры

### Советы для начинающих
1. Старайтесь использовать бонусные клетки
2. Размещайте дорогие буквы (Ф, Щ, Ъ) на клетки с множителем
3. Помните про бонус +50 за использование всех 7 букв
4. Не бойтесь менять буквы, если не можете составить слово
5. Блокируйте бонусные клетки, чтобы противник их не использовал

### Продвинутые стратегии
1. Создавайте несколько слов за один ход (основное + побочные)
2. Планируйте ходы на 2-3 хода вперед
3. Оставляйте себе возможности для следующих ходов
4. Контролируйте доступ к множителям x3
5. Используйте короткие слова для создания зацепок
